# CSV Data Analysis Using Pandas

## Objective
Analyze student data using Pandas and visualize it using Matplotlib and Seaborn.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Loaded CSV dataset
- Calculated average marks
- Found maximum and minimum marks
- Created Bar Chart
- Created Scatter Plot
- Created Heatmap

## Observations
1. Students who study more hours generally score higher marks.
2. Highest marks = 95
3. Lowest marks = 70
4. Heatmap shows positive correlation between Study Hours and Marks.
